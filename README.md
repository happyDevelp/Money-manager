# Money Manager

Welcome to the Animal Handbook, a place where you can learn everything about animals!

## Overview

it makes it possible to read descriptions of animals from all over the world. In other words, it is an encyclopedia of animals.

## Features

Convenient Search: Search for transactions by name or date.
Income and Expense Tracking: Add, edit, and delete income and expense entries.
Monthly Summaries: View monthly summaries of your financial activity.
Favorites: Mark important transactions as favorites for quick access.
Smooth Transitions: Enjoy smooth animations between screen transitions.

- **Income and Expense Tracking: Add, edit, and delete income and expense entries.
- **Favorites: Mark important transactions as favorites for quick access.
- **Convenient Search: Search for transactions by name, date and other.
- **Smooth Transitions:** Enjoy smooth animations between screen transitions.


## How to Use

1. Launch the app.
2. Dashboard: The first screen you see will be the dashboard showing your current balance, total income, and total expenses.
3. Add Transactions: Use the "Add" button to record a new income or expense.
4. View Transactions: Navigate to the transactions list to see all your financial activities.
5. Favorites: Add important transactions to your favorites for quick access.
